#The journey of all 6 levels
Each level builds on the one before it.

Level 1_The Magic of Words and Numbers. Where it all begins: making the computer talk with print(), and storing information inside variables.

Level 2_A Dialogue with the Machine. Programs that listen. Asking the user questions with input(), and doing math with their answers.

Level 3_The Crossroads of Logic. Teaching programs to make decisions using if, elif, else, and logical operators.

Level 4_The Power of Repetition. Letting the computer do the repetitive work for us, with for and while loops.

Level 5_The Alchemist's Lab. Packaging code into reusable functions, so we stop copying and pasting the same lines.

Level 6_The Grand Challenge. Putting it all together into real little programs: a number-guessing game, a simple calculator, FizzBuzz, and a choose-your-own-adventure story.
